Introduction

HTML stands for HyperText Markup Language. It is a mark language that provides structure and content to a webpage.

HTML Basics

HTML is made up of various types of tags. Basic ones include:

  • "h1" is how you make a header.
  • "p" is where you enter your text.
  • "br" is how you add a break inbetween paragraphs or any type of media.
CSS

CSS stands for Cascading Style Sheets. CSS provides style and layout to a webpage. CSS requires a selector followed by curly brackets followed by the style, like this: body{ background-color: #000000; } This is how you would change the background color of your webpage to black.

CSS Images
CSS is most helpful when working with images in a webpage. With CSS you can change the position of an image, you can round the corners, you can make the image into a link, and much more. img{ display: block; margin-left: auto; margin-right: auto; width: 50%; border-radius: 10px; } This is how you would round the corners of your image and center it.
Media Queries

Media Queries are a big part of responsive web pages. With media queries you can add breakpoints. Breakpoints are the “point” at which a website's content and design will adapt in a certain way in order to provide the best possible user experience.

@media(max-width: 480px){ h1{ color: orange; } } @media(max-width: 768px){ h1{ color: yellow; } }

This will make sure that the webpage will adjust the image size for any kind of screen, phone, tablet, or laptop.
